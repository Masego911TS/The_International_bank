# Customer International Payments Portal
A secure full-stack web application for managing international payments, featuring a React frontend and a Node.js and MongoDB backend with advanced security controls.

## Features
1. Customer Authentication – Register, Login, JWT sessions

2. Employee Dashboard – Manage and verify payments

3. SWIFT Payments – Secure payment creation and tracking

## Project Structure

backend/      Node.js + Express + MongoDB
webapp/     React web app (customer and employee dashboards)

## Backend Setup
cd backend
npm install
npm start

## Frontend Setup
cd webapp
npm install
npm start
